# Personality Generator Plugin

**Version**: 3.0.0

交互式人格模型生成器 - AI引导的7轮自然对话构建专属人格

---

## ✨ 核心特性

🤖 **AI主动引导** - 不再需要命令行参数，AI友好开场
💬 **7轮对话流程** - Round 0（关键词） + Rounds 1-6（人格问答）
🎭 **自然对话体验** - "嘿~ 我们来创建你的专属AI人格吧！"
🧹 **自动清理** - 完成后仅保留System Prompt，删除中间文件
⏱️ **2-3分钟完成** - 优化流程，快速生成

## 🚀 快速开始

### 1. 启动生成向导

```bash
/start-wizard
```

AI会主动引导你：

```
嘿~ 我们来创建你的专属AI人格吧！

先告诉我一个关键词，可以是：
  • 角色类型（傲娇猫娘、冷静侦探）
  • 历史人物（诸葛亮、居里夫人）
  • 职业角色（专业医生、温暖老师）
  • 抽象概念（严谨、温暖、专业）

你想要什么样的AI？
```

### 2. 输入关键词

输入你想要的人格类型，AI会分析并给出建议：

```
嘿~ 我懂了！"傲娇猫娘"是经典的角色类型呢~

根据这个关键词，我推荐：

🎭 人格风格: 傲娇型
   外表冷静有个性，但关键时刻超靠谱！

💬 说话方式: 简洁直接
   不啰嗦，直接说重点

❤️  关系定位: 友好同事
   保持点距离感，但你需要帮助时绝对在

这些建议怎么样？接下来我们通过6个问题来确认和调整~
```

### 3. 完成6轮问答

**Batch 1** (4个基础问题):
- Round 1: 人格原型
- Round 2: 表达方式
- Round 3: 情感距离
- Round 4: 纠错方式

**Batch 2** (2个行为问题):
- Round 5: 主导性
- Round 6: 禁忌行为（多选）

每完成一批，AI会给予鼓励：
```
✓ 前4个问题完成了！
你的选择：
  🎭 人格: 傲娇型
  💬 表达: 逻辑完整
  ❤️  关系: 关心朋友
  ✏️  纠错: 直接指出

看起来不错~ 继续最后2个问题吧！
```

### 4. 自动生成

问答完成后，系统自动生成人格模型：

```
🧹 清理中间文件...
  ✓ 已删除会话状态文件
  ✓ 已删除临时JSON配置

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✨ 你的AI人格模型已生成！
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📄 生成文件:
   ~/.claude/personality-models/session-<id>.md

🚀 如何使用:
   方式1 (项目配置):
   cat ~/.claude/personality-models/session-<id>.md >> ./CLAUDE.md

   方式2 (全局配置):
   cat ~/.claude/personality-models/session-<id>.md >> ~/.claude/CLAUDE.md
```

## 📚 常用命令

### 启动向导
```bash
/start-wizard
```
开始创建新的AI人格

### 查看已生成的模型
```bash
/list-personalities
```
浏览所有已生成的人格模型

### 预览模型
```bash
/preview-personality <session-id>
```
查看模型详细配置（应用前预览）

### 应用模型
```bash
# 应用到当前项目（默认）
/apply-personality <session-id>

# 应用到全局配置
/apply-personality <session-id> --global
```
自动将人格模型应用到配置文件

## 🔧 故障排除

### 问题: 脚本权限错误

**解决方案**:
```bash
cd plugin/generate-personal-module
chmod +x scripts/*.sh
```

### 问题: jq命令不存在

**解决方案**:
```bash
# macOS
brew install jq

# Ubuntu/Debian
sudo apt-get install jq
```

### 问题: 找不到生成的模型

**解决方案**:
```bash
# 查看所有已生成的模型
/list-personalities

# 检查生成目录
ls -la ~/.claude/personality-models/
```
