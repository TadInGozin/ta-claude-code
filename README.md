# ta-claude-code

TaTa 的 AI 仓库 - Claude Code 插件市场

## 使用方法

### 添加市场

```bash
# 使用 GitHub（推荐）
/plugin marketplace add TadInGozin/ta-claude-code

# 或使用 Git URL
/plugin marketplace add https://github.com/TadInGozin/ta-claude-code.git
```

### 安装插件

```bash
# 安装指定插件
/plugin install 插件名@ta-claude-code

# 浏览可用插件
/plugin
```

## 可用插件
